# Titanic Survival Prediction using Data Science

## Project Overview

This project was completed as part of the BCS 404 – Introduction to Data Science with Python course.

The objective is to perform Exploratory Data Analysis (EDA), statistical analysis, and machine learning using the Titanic dataset obtained from Kaggle.

## Tools Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Dataset

Titanic Machine Learning Competition Dataset from Kaggle.

## Project Tasks

* Data Acquisition
* Data Cleaning
* Data Visualization
* Statistical Analysis
* Logistic Regression Classification
* Model Evaluation

## Machine Learning Performance

Model: Logistic Regression

Accuracy: **81.01%**
